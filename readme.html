Pixel-Knife GF 像素刻刀高分卫星版

Pixel-Knife 高分卫星 配准 融合

面临问题

高分系列卫星目前已被各行各业广泛接纳，但是使用起来还不太方便，图像质量不高。这主要是因为卫星影像和数码相机不同，在采集的时候是分别形成高分辨率黑白和低分辨率彩色两个数据集，要得到高分的彩色多波段图像需要精细的几何校正和融合复原。目前主流商业软件的处理结果都不尽如人意。

软件简介

“Pixel-Knife GF 像素刻刀高分卫星版” 是Pixel-Knife像素刻刀工作室专门针对高分专项系列卫星的PMS传感器集成开发的的自动精细配准和融合复原的高效能工具集。

通过这个工具集可以高质高效地处理：

  • 2.0米的全波段GF1产品
  • 0.8米的全波段GF2产品
  • 2.0米的全波段GF6产品（卫星待发射，未来跟进）

产品的质量显著地超过目前主流商业遥感软件的质量，而且对操作人员来说没有任何要求，是一个傻瓜化的处理系统。

硬件环境

内存是是一个关键要求：处理GF2要求至少8GB内存。
适当注意硬盘空间：每个GF1融合结果为2GB，每个GF2融合后的为6GB。
估计执行时间（在普通PC或笔记本）：GF1一景7分钟、GF2一景18分钟。

软件环境

本软件在64位的WINDOWS系统上工作。
本软件是绿色软件，就是不需要复杂的安装卸载步骤，用户只需要将安装包直接解压在指定的位置即可。
然后点击pkShell.bat就可以进入命令行环境展开工作。

运行命令

pkRegDemo -root xxx [-fuse] [-cmp]

命令行参数：

-root xxx 指示解压后TIFF文件所在的目录，如果是当前目录就是简单的一个"."
-fuse  指示在精细几何校正后融合复原
-cmp  指示制作只经过RPC投影未经精细几何校正的融合结果，作为几何校正所起作用的参考

例如如下的命令，进入解压后的

cd GF1_PMS2_E119.7_N33.8_20160617_L1A0001647674
pkRegDemo -root . -fuse -cmp

输出文件

LRPAN.TIF
低分辨率的PAN，通常作为配准的基准参考。
LRMS_PROJ.TIF
未精细配准的低分辨率MS，但是已经从原MS上RPC投影到LRPAN坐标系上。
LRMS_REG.TIF
精细配准后的低分辨率MS。
movemap.tif
校正前后的位移幅度图，越亮表示偏移距离越大，色彩越鲜艳表示不同波段之间的偏离程度越大（也就是MS几个波段之间偏差大）。
HRMS_REG.TIF
精细配准后的融合结果，也是这个程序最终的结果。命令行添加-fuse之后才会有。
HRMS_PROJ.TIF
不经过精细配准就融合的结果。命令行添加-cmp之后才会有。

结果比较

  1. 配准效果
    比较[LRMS_UNR / LRPAN]和[LRMS_REG / LRPAN]两组图像之间是否有位移，可以用movemap作为指示，越亮的的地方差别越大

  2. 融合复原效果
